# Digital Business Card

A simple personal business card built with HTML and CSS.

## Preview
This project displays a dark-themed card with:
- Profile photo
- Name and role
- Contact details
- Gradient accent styles

## Tech Stack
- HTML5
- CSS3

## Project Structure
- `index.html`: Card markup and content
- `styles.css`: Layout and styling
- `images/Chafik.jpeg`: Profile image

## Run Locally
1. Open the project folder.
2. Open `index.html` in your browser.
